### 说明：
安装 KubeSphere 3.2.0 的建议 Kubernetes 版本：v1.19.x、v1.20.x、v1.21.x 或 v1.22.x（实验性支持）。
如果不指定 Kubernetes 版本，KubeKey 将默认安装 Kubernetes v1.21.5。
